# android_opengl_game

시작하는 개발자를 위한 OpenGL ES 2.0 안드로이드 게임 프로그래밍 책을 기반으로 프로젝트 진행.


활용 기술 

-OpenGl과 Surface

-Bitmap 랜더링

-파이어베이스

-자동 경로 추적 알고리즘
